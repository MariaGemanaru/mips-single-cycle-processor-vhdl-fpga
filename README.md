# mips-single-cycle-processor-vhdl-fpga
Design and implementation of a simplified MIPS32 single-cycle processor in VHDL for FPGA, including instruction fetch, decode, execute, memory, and control unit modules.
## Architecture
The processor implements the main stages of a single-cycle MIPS architecture:
- Instruction Fetch (IF)
- Instruction Decode (ID)
- Execute (EX)
- Memory Access (MEM)
- Control Unit

## Technologies
- VHDL  
- FPGA design  
- Vivado  
- MIPS architecture
